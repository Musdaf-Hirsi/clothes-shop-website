# KARAMA - Musdaf Shop

A modern clothing shop website built with React, TypeScript, Vite, and Tailwind CSS.

## Live Site

- https://clothes-shop-swart.vercel.app

## Getting Started

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

## Deploy

This project is configured for Vercel. To deploy manually:

```bash
npx vercel --prod
```

## SEO

- `public/sitemap.xml`
- `public/robots.txt`
- Meta description set in `index.html`
